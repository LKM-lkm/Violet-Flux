# 博客主页紫色主题重设计需求

## 项目概述
根据提供的紫色调色板重新设计博客主页，保持浪漫、朦胧、神秘的视觉风格，同时确保布局精确稳定，符合 Nuxt 标准。

## 设计目标
- 使用紫色调色板创建统一的视觉语言
- 保持现有的毛玻璃效果和流体背景
- 优化布局的精确性和稳定性
- 保留浪漫朦胧神秘的氛围
- 确保响应式设计和可访问性

## 调色板
```
主色调：
- #b497d7 (31.88%) - 浅紫色，用于主要背景和强调
- #a682cf (23.32%) - 中紫色，用于次要元素
- #c2a9e4 (16.4%)  - 淡紫色，用于高亮和悬停状态

辅助色：
- #9163c0 (9.93%)  - 深紫色，用于主要按钮和链接
- #a087b4 (9.38%)  - 灰紫色，用于文本和边框
- #7743a3 (4.55%)  - 深紫色，用于重要元素

深色调：
- #837081 (3.37%)  - 深灰紫，用于阴影和深色背景
- #4c2c4f (1.19%)  - 最深紫，用于暗色模式和对比
```

## 技术要求

### 设计系统
- 使用 CSS 变量定义所有颜色
- 创建统一的间距系统（4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px）
- 定义标准的圆角（4px, 8px, 12px, 16px, 24px）
- 统一的阴影系统（sm, md, lg, xl, 2xl）
- 标准的过渡时间（150ms, 200ms, 300ms, 500ms）

### 性能
- 使用 CSS transform 和 opacity 实现动画
- 避免重排和重绘
- 优化毛玻璃效果的性能
- 图片懒加载
- 代码分割

### 可访问性
- 所有交互元素可键盘访问
- 适当的 ARIA 标签
- 颜色对比度符合 WCAG AA 标准
- 焦点状态清晰可见
- 屏幕阅读器友好

## 实现范围

### 需要更新的文件
1. `app/assets/design-system.css` - 更新颜色变量
2. `app/pages/blog/index.vue` - 重新设计博客列表页
3. `app/pages/blog/[...slug].vue` - 更新文章详情页
4. `app/components/ContentSearch.vue` - 更新搜索组件
5. `app/components/ArticleNavigation.vue` - 更新翻页组件
6. `tailwind.config.ts` - 更新 Tailwind 配置

### 不在范围内
- 首页和关于页面（保持现有设计）
- 评论系统样式
- 管理后台

## 设计原则

1. **一致性**: 所有元素使用统一的紫色调色板
2. **层次**: 通过颜色深浅和透明度建立视觉层次
3. **呼吸感**: 充足的留白和间距
4. **流动性**: 平滑的过渡和动画
5. **神秘感**: 使用渐变、光晕和模糊效果
6. **精确性**: 像素级对齐和一致的间距
